# ✨ Overview

The **Contact Management System** is a menu-driven Java console application that allows users to efficiently manage contacts using **Object-Oriented Programming (OOP)** principles.

It demonstrates real-world Java concepts including:

* 📌 Encapsulation
* 📌 Collections Framework
* 📌 CRUD Operations
* 📌 Exception Handling
* 📌 Searching & Sorting
* 📌 Menu Driven Programming
